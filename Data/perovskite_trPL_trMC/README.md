# Reference perovskite trPL and trMC data

This folder contains the reference data for the perovskite trPL and trMC measurements.
The data is from the following publication:

Christian Kupfer, Vincent M Le Corre, Chaohui Li, Larry Lüer, Karen Forberich, Masashi Kato, Andres Osvet, Christoph J Brabec, Unravelling material properties of halide perovskites by combined microwave photoconductivity and time-resolved photoluminescence spectroscopy, Journal of Material Chemistry C, 2024, 12, 95-102, https://doi.org/10.1039/D3TC03867J
