# Reference perovskite trPL
This folder contains the reference data for the perovskite trPL measurements.
The data is from the following publication:


M. Kober-Czerny, A. Dasgupta, S. Seo, F.M. Rombach, D.P. McMeekin, H. Jin, H.J. Snaith, Determining Material Parameters of Metal Halide Perovskites Using Time-resolved Photoluminescence, PRX Energy, 2025, 4, 1, 013001, https://doi.org/10.1103/PRXEnergy.4.013001 

One can also see the GitHub repository for the publication at:
https://github.com/manuelkoberczerny/assessing-TRPL-with-bayesian-inference_and-MCMC
